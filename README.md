# Shared Patch Collecting Data

There are various elements in patch collecting that are shared and this is an open resource for you to use and contribute to for database work related to Boy Scout Patch Collecting. Some items are shared, but some items are unique to certain catalogs. Sort accordingly!

## Contributing

* Fork it
* Make your changes
* Submit a pull request

Pretty straight forward. Please keep things in universal formats such as JSON. If you'd like to also add a SQL version, you may, but just annotate what dialect of SQL you're using and do this in addition to a base JSON file.
